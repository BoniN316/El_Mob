Имеется 2 дизайна
- десктопная elakorD
- мобильная elakorM
Хранятся в файлах template.php
Кроме того, имеются 2 разные морды:
Главная index id=1 - мобильная морда
Главная index1 id=127 - десктопная морда
 Подключение темплат:
  mg.php стр. 641-682 по результатам работы скрипта php определяется тип юзера - мобильный $mob_vibor=1;
  или десктопный/не мобильный/ $mob_vibor=0;
  соответстве6нно устанавливаются пути для получения файла template.php для своего типа юзера.

  Подключение морд:
  производится из файла template.php в строках ~ 314     $main = MG::get('pages')->getPageByUrl('index1');
  соответственно в мобильной версии устанавливается index, в десктопной index1

Скрипты морд хранятся в БД. Доступ через админпанель->Страницы->Главная. Верхняя - мобильная, 2)- десктопная.

  В админке должны быть включены для покавза обе морды Главная
  Ненужная строка Главная в левом меню гасится в файле layout_topmenu.php из десктопного каталога elakorD в строках 38-42 по результатам работы php-скрипта, определяющего тип юзера.
Основной css взагружаемых файлах css
Для мобильной версии style.css
Для десктопной версии style1000.css
Все стили подгружаются из соответствующих темплат.
Кроме основных стилей темплаты подгружают дополнительные стили под соответствующую ширину браузеров.
Выбор - через js механизм прослушивания.
Параметры, изменяемые при переходе от одной ширины к другой корректируются или высчитываются  jsскриптом по данным, сведенным в табличные массивы.
Принцип: в css -оформительская часть,в js - координатно-размерная часть элемента/объекта.
